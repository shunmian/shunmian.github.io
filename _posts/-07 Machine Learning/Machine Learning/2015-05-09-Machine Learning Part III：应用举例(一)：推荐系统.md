## 3 总结 ##

本文介绍了机器学习的一个重要应用，**推荐系统**。

**推荐系统**通过协同过滤算法，可以同时学习用户的喜好$\Theta$和电影的特征$X$。即使刚开始只有少量数据，随着用户的增加和电影的评分增加，$\Theta$和$X$就会越来越准确。
